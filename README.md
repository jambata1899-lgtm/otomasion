# سیستم پیگیر نامه‌های اتوماسیونی — Scaffold

این مخزن یک پروژه‌ی اسکِلفولد Windows Forms (C# / .NET Framework 4.8) است که ویژگی‌های پایه‌ای خواسته‌شده را پیاده‌سازی می‌کند:
- فرم اصلی با DataGridView
- فرم افزودن/ویرایش نامه
- ذخیره و بارگذاری اکسل با ClosedXML
- نگهداری پیوست‌ها در Documents\PaygirLetters_Attachments\[Id]\

⚠️ **نکته مهم درباره‌ی فایل اجرایی (EXE)**  
من اینجا (در محیط این چت) نمی‌توانم به‌طور ایمن یا قابل‌اعتماد پروژه‌های Visual Studio را بدرستی کامپایل و یک EXE تولید کنم. بنابراین:
- من فایل‌های سورس، اسکریپت ساخت (`build_publish.ps1`) و همه‌ی منابع لازم را برای شما آماده کرده‌ام.
- برای تولید EXE باید پروژه را در سیستم ویندوزی که Visual Studio یا msbuild و .NET Framework 4.8 نصب دارد، باز و بیلد کنید. دستورالعمل در `BUILD_INSTRUCTIONS_WINDOWS7.md` آمده است.

اگر می‌خواهید، می‌توانم:
- فایل ZIP شامل سورس را همین‌جا آماده کنم و لینک دانلود بفرستم.
- یا راهنمای مرحله‌به‌مرحله برای بیلد در سیستم شما بنویسم.
